CorTexT Lite is a streamlined, web-based tool designed to transform academic metadata into visual knowledge maps. Inspired by the **CorTexT Manager** platform, this application allows researchers to upload `.ris` bibliographic files and instantly visualize co-authorship and socio-semantic networks.

## ðŸŒŸ Features
* **RIS Parsing:** Supports standard `.ris` exports from Zotero, Scopus, and Web of Science.
* **Dynamic Network Mapping:** Uses force-directed layouts to map relationships between authors and keywords.
* **Interactive Filtering:** Adjust "Connection Strength" in real-time to filter out noise and highlight core research clusters.
* **Deployment Ready:** Optimized for one-click deployment via Streamlit Community Cloud.
